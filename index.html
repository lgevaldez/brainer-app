Brainer Desktop Setup

Wizard para configurar Brainer, Docker, MCP y modelos locales.

1) Preflight

Verifica prerequisitos. Brainer App no instala Docker automáticamente; solo valida y te guía.

2) Workspace Maestro

Selecciona la carpeta raíz (ejemplo: /Users/<you>/Sites). Se montará como /workspaces.

3) GitHub Webhooks

GitHub Token: permite a Brainer publicar comentarios/revisiones en PR.
Webhook Secret: valida firma HMAC y evita eventos falsificados.
Webhook URL: URL que registrarás en GitHub (referencia para ti).

4) Modelo Local (Ollama)

Se detectan modelos instalados. Si no tienes uno, puedes instalar uno recomendado.
