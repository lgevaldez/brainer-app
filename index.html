Brainer Operator

Cliente desktop oficial para instalar, configurar y operar brainer.

1) Preflight

Verifica prerequisitos. La app no instala Docker automáticamente; solo valida y te guía para operar brainer.

Si la autodetección falla, selecciona manualmente la carpeta del repo brainer.

2) Workspace Maestro

Selecciona la carpeta raíz (ejemplo: /Users/<you>/Sites). Se montará como /workspaces.

3) GitHub Webhooks

GitHub Token: permite a brainer publicar comentarios/revisiones en PR.
Webhook Secret: valida firma HMAC y evita eventos falsificados.
Webhook URL: URL que registrarás en GitHub (referencia para ti).

4) Modelo Local (Ollama)

Se detectan modelos instalados. Si no tienes uno, puedes instalar uno recomendado.

5) MCP Agents

Selecciona en qué clientes deseas instalar la integración MCP de brainer.

La instalación MCP se ejecuta después de levantar el stack.
